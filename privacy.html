Политика конфиденциальности

Дата последнего обновления: 24 марта 2026 г.

1. Общие положения

Настоящая Политика конфиденциальности (далее — Политика) определяет порядок обработки и защиты персональных данных физических лиц, которые предоставляют свои данные при использовании сайта rubikonbtp.ru (далее — Сайт).

Оператором персональных данных является ООО «Рубикон» (ИНН 7811814395, КПП 781101001), зарегистрированное по адресу: 193318, г. Санкт-Петербург, ул. Ванеева, д. 10, лит. Б, пом. 3-Н, офис 308-309 (далее — Оператор).

Заполняя формы на Сайте и/или используя голосовой чат, Пользователь выражает согласие с условиями настоящей Политики.

2. Цели обработки персональных данных

Оператор обрабатывает персональные данные Пользователя в следующих целях:

  • Обработка входящих запросов и заявок на расчёт оборудования
  • Подготовка и направление коммерческих предложений
  • Связь с Пользователем по телефону, электронной почте или через мессенджеры
  • Улучшение качества обслуживания и работы Сайта

3. Перечень обрабатываемых персональных данных

Оператор может обрабатывать следующие персональные данные Пользователя:

  • Фамилия, имя, отчество
  • Номер контактного телефона
  • Адрес электронной почты
  • Наименование организации
  • Содержание обращений и заявок
  • Данные, получаемые при использовании голосового чата (аудиозаписи обращений)
  • Данные об IP-адресе, типе браузера, времени доступа (автоматически)

4. Правовые основания обработки

Обработка персональных данных осуществляется на основании:

  • Согласия субъекта персональных данных (п. 1 ч. 1 ст. 6 Федерального закона № 152-ФЗ)
  • Необходимости исполнения договора, стороной которого является субъект (п. 5 ч. 1 ст. 6 ФЗ-152)

5. Порядок обработки персональных данных

Обработка персональных данных осуществляется с использованием средств автоматизации и без таковых, включая сбор, запись, систематизацию, накопление, хранение, уточнение, извлечение, использование, передачу, обезличивание, блокирование, удаление и уничтожение.

Оператор принимает необходимые организационные и технические меры для защиты персональных данных от неправомерного доступа, уничтожения, изменения, блокирования, копирования, распространения.

6. Хранение персональных данных

Персональные данные хранятся на серверах, расположенных на территории Российской Федерации. Срок хранения персональных данных — до момента отзыва согласия субъектом персональных данных или до истечения 3 (трёх) лет с момента последнего обращения Пользователя.

7. Передача персональных данных третьим лицам

Оператор не передаёт персональные данные третьим лицам, за исключением случаев:

  • Наличие согласия Пользователя
  • Требование законодательства Российской Федерации
  • Необходимость для исполнения заключённого договора

8. Использование файлов cookie и аналитика

Сайт использует файлы cookie и сервисы веб-аналитики (Яндекс.Метрика) для сбора статистических данных о посещении Сайта. Данные инструменты могут автоматически собирать информацию о действиях Пользователя на Сайте (посещённые страницы, время визита, источник перехода). Эти данные используются в обезличенном виде для улучшения работы Сайта.

9. Голосовой чат (AI-менеджер)

На Сайте используется голосовой AI-менеджер для консультирования Пользователей. При использовании голосового чата:

  • Аудиозаписи речи Пользователя обрабатываются с использованием сервиса распознавания речи Яндекс SpeechKit для преобразования в текст
  • Текстовое содержание диалогов может сохраняться для целей обработки заявок
  • Контактные данные, озвученные Пользователем в ходе диалога, обрабатываются в соответствии с настоящей Политикой

10. Права субъекта персональных данных

Пользователь имеет право:

  • Получить информацию об обработке своих персональных данных
  • Потребовать уточнения, блокирования или уничтожения персональных данных
  • Отозвать своё согласие на обработку персональных данных

Для реализации указанных прав направьте запрос на info@kbbcompany.ru или по адресу: 193318, г. Санкт-Петербург, ул. Ванеева, д. 10, лит. Б, пом. 3-Н, офис 308-309.
